<p>To unit test my JavaScript files in Microsoft Visual Studio 2012, side by side mine C# unit tests, I use Chutzpah.</p>  <p>Chutzpah comes with 2 Microsoft Visual Studio 2012 extensions:</p>  <p>1. Chutzpah Test Adapter for Visual Studio 2012, allows you to run QUnit / Jasmine javascript test files side by side C# unit tests in the test explorer or continuous build process.</p>  <p>2. Chutzpah – A JavaScript Test Runner, allows you to run individual QUnit / Jasmine JavaScript test files in Microsoft Visual Studio 2010 and 2012 by right clicking a test file and choosing &quot;&quot;.</p>  <p>&#160;</p>  <p><strong>In Microsoft Visual Studio 2012</strong></p>  <p>- Tools &gt; Extensions and Updates…</p>  <p>- Install both plugins and restart Microsoft Visual Studio 2012</p>  <p>&#160;</p>  <p><a href="http://www.roelvanlisdonk.nl/wp-content/uploads/2012/11/image5.png" rel="lightbox"><img title="image" style="border-left-width: 0px; border-right-width: 0px; background-image: none; border-bottom-width: 0px; padding-top: 0px; padding-left: 0px; margin: 0px 5px; display: inline; padding-right: 0px; border-top-width: 0px" border="0" alt="image" src="http://www.roelvanlisdonk.nl/wp-content/uploads/2012/11/image_thumb5.png" width="580" height="400" /></a></p>  <p>&#160;</p>  <p>Create a new UnitTest project:</p>  <p>&#160;</p>  <p><a href="http://www.roelvanlisdonk.nl/wp-content/uploads/2012/11/image6.png" rel="lightbox"><img title="image" style="border-left-width: 0px; border-right-width: 0px; background-image: none; border-bottom-width: 0px; padding-top: 0px; padding-left: 0px; margin: 0px 5px; display: inline; padding-right: 0px; border-top-width: 0px" border="0" alt="image" src="http://www.roelvanlisdonk.nl/wp-content/uploads/2012/11/image_thumb6.png" width="580" height="465" /></a></p>  <p>&#160;</p>  <p><strong>Enable &quot;Run Tests after build&quot;</strong></p>  <p>(The &quot;Play&quot; button found at the top of the Test Explorer&quot;)</p>  <p>Now when you press CTRL + SHIFT + B the project will be build and all tests will execute.</p>  <p>&#160;</p>  <p><a href="http://www.roelvanlisdonk.nl/wp-content/uploads/2012/11/image7.png" rel="lightbox"><img title="image" style="border-left-width: 0px; border-right-width: 0px; background-image: none; border-bottom-width: 0px; padding-top: 0px; padding-left: 0px; margin: 0px 5px; display: inline; padding-right: 0px; border-top-width: 0px" border="0" alt="image" src="http://www.roelvanlisdonk.nl/wp-content/uploads/2012/11/image_thumb7.png" width="580" height="347" /></a>&#160;</p>  <p>&#160;</p>  <p><strong>TIP</strong>: Never use CTRL + S to save your file, just use CTRL + SHIFT + B.</p>  <p>Now you can see the C# unit test &quot;TestMethod1&quot; succeeded.</p>  <p>&#160;</p>  <p><strong>QUnit for ASP.NET MVC</strong></p>  <p>Right click unit test project and click Manage NuGet Packages…</p>  <p>Click install.</p>  <p>&#160;</p>  <p><a href="http://www.roelvanlisdonk.nl/wp-content/uploads/2012/11/image8.png" rel="lightbox"><img title="image" style="border-left-width: 0px; border-right-width: 0px; background-image: none; border-bottom-width: 0px; padding-top: 0px; padding-left: 0px; margin: 0px 5px; display: inline; padding-right: 0px; border-top-width: 0px" border="0" alt="image" src="http://www.roelvanlisdonk.nl/wp-content/uploads/2012/11/image_thumb8.png" width="580" height="546" /></a></p>  <p>&#160;</p>  <p><strong>Add Code.js and Test.js</strong></p>  <p>The Code.js file will contain the JavaScript code we want to test.</p>  <p>The Test.js&#160; file will contain the JavaScript test code written, by using the QUnit test framework.</p>  <p>&#160;</p>  <p><strong>Code.js</strong></p>  <p><a href="http://www.roelvanlisdonk.nl/wp-content/uploads/2012/11/image9.png" rel="lightbox"><img title="image" style="border-left-width: 0px; border-right-width: 0px; background-image: none; border-bottom-width: 0px; padding-top: 0px; padding-left: 0px; margin: 0px 5px; display: inline; padding-right: 0px; border-top-width: 0px" border="0" alt="image" src="http://www.roelvanlisdonk.nl/wp-content/uploads/2012/11/image_thumb12.png" width="580" height="244" /></a></p>  <p>&#160;</p>  <p><strong>Run only JavaScript unit test file</strong></p>  <p>If you only want to run the QUnit test, right click on the JavaScript Test.js file and choose &quot;Run JS Tests&quot;.</p>  <p>&#160;</p>  <p><a href="http://www.roelvanlisdonk.nl/wp-content/uploads/2012/11/image13.png" rel="lightbox"><img title="image" style="border-left-width: 0px;
